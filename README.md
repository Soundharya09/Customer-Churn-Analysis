# 📊 Customer Churn Analysis & Dashboard (Power BI + Python)
## 📌 Project Overview
This project focuses on analyzing customer churn behavior using Python (EDA) and Power BI (interactive dashboarding) to generate actionable 
business insights. The goal is to help organizations identify at-risk customers, understand churn drivers, and improve retention strategies 
through data-driven decision-making.

## 🎯 Business Objective
- Identify key factors driving customer churn
- Segment high-risk customers
- Provide actionable insights to reduce churn
- Build an interactive dashboard for stakeholders

## 🔍 Analysis Workflow

### 1. Data Cleaning (Python)
- Handled missing values
- Converted categorical variables
- Prepared dataset for analysis
  
### 2. Exploratory Data Analysis (EDA)
- Customer segmentation
- Churn distribution
- Feature relationships
  
### 3. Dashboard Development (Power BI)
Created an interactive dashboard with 3 main pages:

### 📊 Dashboard Insights
#### 📌 1. Executive Overview
- Total customers, churn rate, and revenue impact
- High-level KPIs for decision-makers

#### 👥 2. Customer Behavior Analysis
- Churn by contract type
- Churn by payment method
- Service usage patterns
- 📌 Key Finding: Customers with monthly contracts have significantly higher churn

#### ⚠️ 3. Risk & Retention Insights
- High-risk customer segments
- Revenue loss due to churn
- Retention opportunities

#### 📌 Key Findings:
- Customers using electronic check payments show higher churn
- Short-tenure customers are more likely to leave
- Higher monthly charges correlate with increased churn

## 💡 Business Recommendations
- Encourage long-term contracts with incentives
- Target new customers early with engagement strategies
- Improve experience for high-value customers
- Optimize payment methods to reduce friction
